### Run Locally

To run **Wren** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/sadhin555/wren-blog.git
```

Windows:

```bash
git clone https://github.com/sadhin555/wren-blog.git
```
